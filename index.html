OpenWeatherMap and Vue.js

This is a simple Vue.js app, fetching weather data for a user-selected location, for educational purposes. It uses the Google Geocode API for location suggestions based on user input or GPS position (if available), and OpenWeatherMap API to render 5 days weather forecast.

I also use Vuex for state management, for no other reason than getting to know the workings of it. HTTP requests are handled with axios. Finally, I use a local, patched version of Vue Select component, with a couple of PRs baked in. It will be updated, eventually.
